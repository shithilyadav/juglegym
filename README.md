### Run Locally

To run **junglegym** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo clone git https://github.com/shithilyadav/juglegym.git
```

Windows:

```bash
git clone https://github.com/shithilyadav/juglegym.git
```
